# Project Brief: Cross-Platform File Manager

## Project Overview

A cross-platform file manager app (except web) focused on media management, tagging, and video streaming, with a tabbed interface and smooth performance.

## Goals & Objectives

- Provide a user-friendly, feature-rich file manager for all major platforms
- Make file tagging and searching easy, especially on Windows
- Enable streaming and video playback directly in the app
- Support convenient file organization and management
- Ensure seamless, consistent experience across devices

## Key Features

- Tagging and searching files by tag
- Video streaming and playback
- Tabbed navigation
- Fast file search and batch operations
- Media preview (images, audio, video)
- Cloud/network integration (optional)
- Customizable UI, secure file management, favorites, drag-and-drop, sharing, advanced filtering, multi-language, accessibility, in-app help, activity log, plugin system

## Target Users & Use Cases

- Designed for anyone needing powerful file/media management
- No specific user group targeted
- Use cases: organizing files, tagging/searching media, streaming/viewing videos, managing files across devices

## Business/Technical Constraints

- Support all Flutter platforms except web (Android, iOS, Windows, macOS, Linux)
- App must be smooth and performant
- Operate primarily as a local app; cloud features are optional
- Prioritize privacy and local data management

## Success Criteria

- App runs smoothly and consistently on all supported platforms
- Tagging and search are intuitive and fast
- Media streaming and playback work reliably
- Users can manage files easily without cloud dependency
- Optional cloud features do not impact local performance
